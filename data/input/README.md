# Sources

- parcels-w-fzp-model-data.csv ([SF City Economist FZP parcels data](https://docs.google.com/spreadsheets/d/1yR1iPH_1OT6zjacUlI-hYuX2t3fpCi7s/edit?usp=sharing&ouid=102702719400652445235&rtpof=true&sd=true)) Last updated Nov 5, 2025. this is the dataset which includes the atributes needed to plug into the prediction model. This data file forms the core and gold truth standard for the data munging processes.
